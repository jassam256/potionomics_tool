# Potionomics Application Planning

## Table

- Add header fields for sorting (ascending, descending)
- Add indicators for ascending, descending, none
- Consistent cell width (column layout)

## Ingredients

Each ingredient consists of:

- ID
- Name
- Location
- Rarity
- Type
- Magimins (A to E)
- Base Price
- Amount sold by Quinn
- Traits (Taste, Sensation, Aroma, Visual)

## Search

Can search an ingredient by its name

## Filter

Can set filter settings for different fields

Some will have toggle settings (on and off)

Magimins will have MIN and MAX values alongside the ability to toggle each one off (e.g. Only A, none that give B etc.)

## Sort

Can sort by each of its attributes:

- ID
- Name
- Location
- Rarity
- Type
- Magimins (A to E)
- Base Price
- Amount sold by Quinn
- Traits (Taste, Sensation, Aroma, Visual)
